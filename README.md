f - **f**ish - fish

g - **g**ood - gòd

h - **h**ood - hòd

i - **i**nn - inn

ì - f**i**ne - fìne

also **c**ake - kâke

l - **l**oom - lòm

m - **m**oon - mòn

n - **n**oon - nòn

ò - d**oo**m - dòm

p - **p**izza - peza

q - **ch**ick - qick

r - **r**at - rat

s - **s**oup - sòp
